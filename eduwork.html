Git Fundamental

  • "git init", untuk menginisialisasi repository
  • "git config", configurasi email dan username
    • git config --local user.name "username"
    • git config --local user.email "email@email.com"
    • git config --global user.email "email@email.com"
  • "git status", untuk mengetahui status file
  • "git add .", untuk memasukkan file ke stagging area
  • "git log", untuk melihat riwayat git commit
  • "git commit", untuk mencatat perubahan yang dilakukan
    • git commit -m "first commit"
  • "git clone", untuk menyalin repo dari server
  • "git push", untuk mengirimkan commit ke server
  • "git remote", interaksi dari client dengan server
  • "gitu pull", mengambil commit dari server jika ada perubahan di server yang dibuat dari branch lain

Git Branch

  • "git branch" , melihat semua daftar branch
  • "git branch -all", melihat semua branch public
  • "git branch nama_branch", membuat branch baru
  • "git checkout nama_branch", berpindah branch
  • "git merge", untuk menggabungkan commit dari 2 branch
    • relatif dengan branch yang aktif saat ini
    • untuk merge harus dipastikan tidak ada file yang belum di commit
  • conflict
    • kondisi yang terjadi ketika merge 2 branch yang memiliki perubahan file yang sama
    • ketika konflik tidak bisa commit
    • untuk commit konflik harus diselesaikan dulu
